# AI 小说生成器 - 前端

Vue 3 + Vite 单页应用，对接后端 `http://localhost:8080`（开发时通过 Vite 代理 `/api`）。

## 功能

- **首页**：入口导航（创建小说、资料库、我的小说）
- **资料库**：上传资料（名称 + 文本），自动向量化入库；列表与删除
- **我的小说**：列表、创建、编辑、删除
- **创建/编辑小说**：标题、题材、世界观设定，**可多选关联资料库**
- **小说详情**：设定、关联资料库、结构、章节列表

## 运行

```bash
# 安装依赖
npm install

# 开发（默认 http://localhost:5173，API 代理到 8080）
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 用户 ID

页头可输入「用户 ID」，用于区分当前用户的小说与资料库（本地会持久化到 localStorage）。
